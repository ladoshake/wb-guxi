# 股息率排名（A股 / 港股 / 美股 · 市值 > 1000 亿 · TTM / LFY 前 30）

覆盖 **A 股、港股、美股** 三个市场，按各市场**本币**独立筛选总市值超过 **1000 亿（元 / 港元 / 美元）** 的公司，按股息率分别取 **TTM（近 12 个月）** 与 **LFY（最近财年）** 排名的前 30 名，生成可交互的网页榜单。

## 功能特性

- **三市场切换**：A 股 / 港股 / 美股，各市场独立筛选、独立榜单；表头币种随市场切换（¥ / HK$ / $）。
- **市值分两档**：各市场内再按市值分两档 —— `市值 > 1000亿` 与 `1000亿 ≥ 市值 > 500亿`，**以各市场本币（元 / 港元 / 美元）为基准独立计算，互不折算**，每档取股息率 Top 30。
- **双榜单切换**：TTM 股息率排名 / LFY 股息率排名。
- **LFY 表附加趋势列**：在「LFY 股息率」之后，额外列出**前一年**与**前两年**股息率（口径同 LFY），便于观察分红连续性；表头标注具体年份（随市场切换）。
- **同财年多次分红累加**：年内若有多次分红（年报 + 中期 + 特别），均按财年/窗口求和。
- **表头点击排序**、**隔行底色**、**悬停高亮**。
- **数据刷新按钮**（需本地服务器，见下方「运行方式」）；纯前端渲染，数据为抓取日快照，可离线打开查看。

## 数据口径

| 指标 | 计算方式 |
|---|---|
| **总市值筛选** | 腾讯自选股（WeStock）实时行情「总市值」，各市场本币、单位「亿」：A股=亿元、港股=亿港元、美股=亿美元；**以各市场本币直接分档，互不折算** |
| **每股分红** | 各市场分红明细的派息比例 ÷ 10 = 元/股（按各市场本币）：A股=`cashDiviRMB`（元/10股，腾讯自选股 dividend list）；港股=`cashDivPerShare`；美股=`dividend` |
| **TTM 股息率** | **三市场统一取行情接口的 TTM 股息率（dividend_ratio_ttm，腾讯自选股 quote）** |
| **LFY 股息率** | 最近一个完整财年的现金分红合计数 ÷ 当前价 × 100%。**A股/港股按分红所属报告期（财年）统计；美股按除息日所在自然年近似**（当年未结束则取上一个完整自然年）。港股/美股历史分红来自腾讯自选股 dividend list（--years 5） |
| **前一年 / 前两年股息率** | 口径同 LFY，分别对应 LFY 财年 −1、−2 的全部分红 |

> 运行日期为动态 `dt.date.today()`，因此 TTM 窗口与「最近财年」随运行日自动更新，每次刷新反映的是当日真实情况。

## 文件说明

| 文件 | 作用 |
|---|---|
| `fetch_data.py` | 抓取 **A 股 / 港股 / 美股** 三市场（均通过腾讯自选股 WeStock：westock-tool filter 筛选股票池、westock-data quote 行情、westock-data dividend list 历史分红），计算 TTM/LFY 股息率，生成 `data.json` |
| `build_html.py` | 读取 `data.json`，生成自包含网页 `index.html` |
| `serve.py` | 本地 HTTP 服务（`stdlib`，无需额外依赖），提供刷新接口 |
| `index.html` | 交付网页（数据内嵌，可直接双击打开） |
| `.gitignore` | 排除运行时缓存与本地目录 |

运行时生成的 `data.json` 不纳入版本管理。

## 运行方式

环境要求：Python 3.10+，依赖 `akshare`、`pandas`、`requests`（建议使用虚拟环境）。

```bash
pip install akshare pandas requests

# 抓取数据 + 生成网页（A股 + 港股 + 美股，约数分钟）
python fetch_data.py
python build_html.py
```

### 仅查看榜单
直接双击打开 `index.html` 即可（使用内嵌的数据快照）。此时刷新按钮会提示需启动服务器。

### 实时刷新数据
启动本地服务器，再用浏览器访问 `http://localhost:8000`：

```bash
python serve.py
```

点击网页右上角「刷新数据」即可重跑抓取脚本并更新榜单。

| 接口 | 说明 |
|---|---|
| `GET /` | 打开 `index.html` |
| `GET /api/data` | 读取最新 `data.json` |
| `POST /api/refresh` | 重跑 `fetch_data.py` + `build_html.py` |

> 手动刷新复用已有分红缓存，仅实时重拉市值并重算（通常数十秒）；新披露的分红由每日定时任务（先清缓存全量重抓）入账。

## 数据来源

- **行情 / 总市值 / 历史分红（港股 / 美股）**：腾讯自选股 **WeStock** 技能（`westock-tool` filter 筛选股票池、`westock-data` quote 实时行情、`westock-data` dividend list 历史分红），与腾讯自选股 App / qt.gtimg.cn 同源
- **行情 / 总市值（A 股）**：腾讯财经 gtimg
- **A 股列表与历史分红**：akshare（巨潮 `stock_dividend_cninfo`）
